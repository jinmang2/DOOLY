# Dooly
배치화를 지원하는 pororo의 친구 둘리!  huggingface로 모든 것을 처리합니다.

때문에 fairseq 등 복잡한 dependency없이 쉬운 사용이 가능해집니다.

windows 환경에서 사용도 가능합니다.

## Support Tasks
- Translation
- Named Entity Recognition
- Natural Language Inference
- Machine Reading Comprehension
